========================================
  运单号自动化 - 部署指南
========================================

## 准备工作

### 1. 目标电脑要求
- Windows 7/10/11
- Chrome 浏览器（必须）
- chromedriver.exe（与 Chrome 版本匹配）

### 2. 打包项目
```bash
pyinstaller kd_auto_service.spec
```

### 3. 需要复制的文件

打包后，把整个项目文件夹复制到目标电脑，包含：

```
项目文件夹/
├── chromedriver.exe       # Chrome 驱动（必须）
├── nssm.exe               # NSSM（可选，做服务需要）
├── dist/
│   └── 运单号自动化工具.exe  # 主程序
├── config/
│   └── config.ini         # 配置文件
└── *.bat                  # 脚本文件
```

## 配置修改

复制到目标电脑后，修改 `config/config.ini`：

```ini
[DATABASE]
# 如果目标电脑能访问原数据库，保持不变
# 否则修改为新电脑能访问的数据库地址

[SERVER]
port = 8000  # 如需更改端口

[PATH]
# 建议改为相对路径或目标电脑的路径
chrome_profile_dir = chrome_profile
screenshot_dir = screenshots
log_dir = logs
```

## 运行方式

### 方式1: 直接运行
```
dist\运单号自动化工具.exe
```
然后浏览器访问 http://localhost:8000

### 方式2: 安装 Windows 服务
```
install_service.bat
```
（需要管理员权限）

## 常见问题

Q: 提示 chromedriver 版本不匹配
A: 下载与目标电脑 Chrome 版本匹配的 chromedriver

Q: 服务启动失败
A: 检查 config.ini 配置是否正确

Q: 数据库连接失败
A: 检查 DATABASE 配置，确保目标电脑能访问数据库
```
